เกี่ยวกับฉัน

นักศึกษาประกาศนียบัตรวิชาชีพ ชั้นปีที่ 3 กำลังศึกษาอยู่ที่วิทยาลัยเทคโนโลยีอักษรพัทยา
สามารถเรียนรู้ได้รวดเร็ว พร้อมปรับตัวให้เข้ากับเทคโนโลยีใหม่ สังคมใหม่อยู่เสมอ
มีทักษะที่ดีในด้านภาษา Java และ Visual Basic Studio พร้อมกับมีประสบการณ์ใน
ด้านการออกแบบ การสร้าง การปรับแต่งเว็บไซต์โดยใช้ HTML, CSS และ JavaScript

กำลังมองหาที่ฝึกงานหรือโอกาสในระดับเริ่มต้น เพื่อได้นำทักษะด้าน
การเขียนโปรแกรมและการพัฒนาเว็บไซต์ไปใช้และพัฒนาต่อยอดในการทำงานจริง

สิ่งที่ฉันทำได้

  • ออกแบบเว็บไซต์

    ออกแบบและพัฒนาเว็บไซต์ที่ใช้งานง่าย รองรับการแสดงผลทุกอุปกรณ์ โดยใช้โครงร่าง สีสัน และตัวอักษรที่ทันสมัย

  • สร้างเว็บไซต์

    พัฒนาเว็บไซต์ด้วย HTML, CSS และ JavaScript โดยเพิ่มความสามารถในการโต้ตอบ และรองรับการแสดงผลบนทุกอุปกรณ์

  • แอปพลิเคชัน Windows Forms

    พัฒนาแอปพลิเคชันบนเดสก์ท็อปด้วย Visual Basic และ Java โดยมีส่วนติดต่อผู้ใช้ (GUI) เช่น ปุ่มและแบบฟอร์ม

  • แอปพลิเคชันมือถือ

    ใช้เครื่องมือพื้นฐานในการสร้างแอปพลิเคชันง่าย ๆ และเรียนรู้โครงสร้างการออกแบบ UI บนมือถือ
